# HyperOS Notify Icon Fix

An LSPosed module that restores the default Android-style notification icons which HyperOS overrides with app icons.

Tested and confirmed working on **HyperOS 3**. Compatibility with other versions is not guaranteed.

This module was made just for fun, so if you find any issues, please let me know via [Issues](https://github.com/KimPig/HyperOSNotifyIconFix/issues).

---

HyperOS가 변경한 알림 아이콘을 기존 안드로이드 방식으로 되돌려주는 LSPosed 모듈입니다.

**HyperOS 3**에서 작동을 확인했으며, 그 외 버전에서의 작동은 보장하지 않습니다.

심심해서 만든 모듈이니, 문제가 있다면 [Issues](https://github.com/KimPig/HyperOSNotifyIconFix/issues)에 알려주세요.
